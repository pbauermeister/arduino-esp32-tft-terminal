# Deprecated

This directory holds the original FeatherWing OLED server
(`oled-server.ino`), superseded by `server-esp32s3-rtft/`.

It is kept for reference only:

- It targets different hardware (an OLED FeatherWing, not the
  ESP32-S3 Reverse TFT Feather).
- It is not built, maintained, or covered by the project tooling
  (Makefile, tests, CI).

Do not extend it. New firmware work belongs in `server-esp32s3-rtft/`.
